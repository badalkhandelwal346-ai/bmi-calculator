🧮 BMI Calculator (React + Vite)

A simple and responsive Body Mass Index (BMI) Calculator built using React and Vite. This project helps users calculate their BMI based on height and weight inputs.

🚀 Features
📏 Input height (in cm) and weight (in kg)
⚡ Instant BMI calculation
🎯 Simple and clean UI
🔄 Fast performance using Vite
📱 Responsive design
🛠️ Tech Stack
React (Frontend library)
Vite (Build tool)
CSS (Styling)
